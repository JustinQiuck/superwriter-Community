# Support

SuperWriter is source-available for personal non-commercial use.

## Community Support

Public issues are best for:

- reproducible bugs;
- documentation corrections;
- small improvement suggestions;
- setup problems that do not expose private data.

Maintainers may not be able to provide guaranteed response times, debugging sessions, deployment help, or one-on-one product support for free personal use.

## Commercial Permission

Commercial, organizational, production, hosted, paid, redistributed, or productized use requires prior written permission. Do not open a public issue with private business details; contact the project maintainer privately.

## Security

Do not report vulnerabilities, leaked secrets, auth bypasses, RLS bypasses, or data exposure in public issues. Follow [SECURITY.md](./SECURITY.md).

## Before Asking

- Check [README.md](./README.md) and [.env.example](./.env.example).
- Search existing issues first.
- Remove API keys, tokens, cookies, private URLs, and user data from logs.
- Include your OS, Node.js version, pnpm version, and relevant command output.
